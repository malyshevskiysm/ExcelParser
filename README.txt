Данная программа позволяет скачивать файл формата .xls с компьютера клиента, парсить его 
и добавлять содержимое в базу данных.

Требования к файлу:
- формат xls;
- три столбца в очередности: организация, адрес, телефон;
- первая строка является заголовком и не заносится в базу данных;

Требования к бд:
- СУБД MySQL;
- имя таблицы organizations;
- таблица должна содержать обязательное поле organization типа varchar;
- таблица должна содержать обязательное поле address типа varchar;
- таблица должна содержать обязательное поле phone типа varchar;
